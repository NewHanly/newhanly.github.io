表

一开头线扔出四张表raw、mangle、nat、filter

表功能
mangle拆解报文做出修改，重新封装
nat网络地址转换功能
filter过滤功能，防火墙
raw关闭nat表上启用的连接追踪机制

链

简单的说“链”就是对一些行为相似的数据包采用的规则集合

链匹配的报文能使用的表
INPUT处理输入数据包mangle、nat*、filter
OUTPUT处理输出数据包raw、mangle、nat、filter
FORWARD处理转发数据包mangle、filter
PREROUTING用于目标地址转换（DNAT）raw、mangle、nat
POSTROUTING用于源地址转换（SNAT）mangle、nat

某些系统规定nat表中的规则不能用于INPUT链

匹配

基本匹配规则
| 参数 | 内容 |
| — | — |
| s | 匹配来源IP地址，“!”表示除外 |
| d | 匹配目标IP地址 |
| i | 网卡名，匹配从该网卡流入的数据包 |
| o | 网卡名，匹配从该网卡流出的数据包 |
| p | 匹配协议,如tcp,udp,icmp |

动作

参数动作
ACCEPT接受
REJECT拒绝
DROP丢弃数据包
RETURN数据包不继续执行该链后面的命令ALL
REDIRECT转发(仅用于nat表)
